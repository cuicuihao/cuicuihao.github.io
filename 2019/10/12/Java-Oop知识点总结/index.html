第三章 面向对象三大特征—封装

  • 封装的概念

  • 封装的步骤

    • 把属性私有化（private）
    • 对这些属性设置get和set方法
    • 调用处理这些方法

  • 封装的作用：避免一些用一些不合乎常理的操作

  • 修饰符

    • public—>公共的：在一个项目中都可以访问，权限最高

    • private—>私有的：只能在本类中进行访问，权限最低

    • protected—>安全的，受保护的：同一个包能进行访问，继承中之类可以访问父类

    • default—>默认的：同一个包能访问

      修饰符 同一个类 同一个包 子类 所有类
      private(私有的) *
      default(常规) * *
      protected(受保护的) * * *
      public(公共的) * * * *
      从小到大优先级:

      private < default < protected < public
